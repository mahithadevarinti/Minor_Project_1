# 🧬 GroupDNA — WhatsApp Group Chat Decoder

GroupDNA is a lightweight, high-performance WhatsApp group chat analytics engine built entirely from scratch. It parses raw, messy chat export files (`.txt`) and generates a comprehensive behavioral and mathematical report directly in your terminal.

This project was built as part of **The Unlox Academy Week 1 Minor Project**.

---

## 🚫 The Constraint Discipline (The Hard Core)
To master true data engineering and algorithmic fundamentals, this project was built under a strict self-imposed constraint:
* **Forbidden:** `pandas`, `matplotlib`, `collections.Counter`, `re` (Regular Expressions)
* **Allowed:** Pure Python standard libraries (`datetime`) + `numpy`

Every parsing mechanic, data aggregation loop, and visualization layout was engineered completely from the ground up without relying on high-level frameworks.

---

## 📊 Key Features & Analytics Tracked

* **Robust Custom Parser:** Reads raw WhatsApp exports line-by-line while handling real-world edge cases like multi-line text continuations, system notifications, deleted messages, and media omission alerts using strict string-slicing logic (`str.split`, `str.strip`).
* **Activity Heatmap (NumPy Matrix):** Builds a 2D matrix ($num\_participants \times 24$ hours) using NumPy arrays to map user activity across time blocks, rendering it as terminal block-art shading (`█`, `▒`, `░`, `.`).
* **Text Normalization & Word Frequencies:** Features a custom tokenizer that strips edge punctuation and filters language-specific stop-words manually to extract top-10 vocabulary frequencies and top-3 personal favorite words[cite: 1].
* **Response Speeds & Silent Streaks:** Calculates consecutive silent streak durations (runs of days with zero activity) and calculates reply time latency down to the exact minute[cite: 1].
* **Greedy Archetype Classification:** Runs a custom classification algorithm that normalizes scores across 9 distinct group behaviors to greedily assign unique friend archetypes (e.g., *The Night Owl*, *The Storyteller*, *The Drama Queen*, *The Group Mom*, *The Spammer*, or *The Ghost*)[cite: 1].

---

## 🛠️ Tech Stack & Concepts Used

* **Languages:** Python 3
* **Libraries:** NumPy (Matrix manipulation, statistical aggregations)
* **Core Concepts:** Complex loops, nested dictionaries for multi-dimensional mapping, object parsing with `datetime.strptime`, matrix arithmetic, sorting algorithms via custom keys, and string formatting layouts[cite: 1].
